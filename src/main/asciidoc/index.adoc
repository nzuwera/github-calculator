== Calculator API
Collection of API endpoints used to perform arithmetic operations on lists of numbers.

=== Perform Operations on a List of Numbers
Perform arithmetic operations (addition, subtraction, multiplication, division) on a list of numbers.

==== Operations
* `A` - Addition: Adds all numbers in the list
* `S` - Subtraction: Subtracts all numbers from the first number
* `M` - Multiplication: Multiplies all numbers in the list
* `D` - Division: Divides the first number by all other numbers

==== POST Endpoint
This endpoint accepts a list of numbers in the request body and performs the specified operation.
